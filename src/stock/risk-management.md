# 风险管理

风险管理是股票投资中最重要的环节，本页面记录我的风险控制方法和经验。

## 风险类型

### 系统性风险
- 宏观经济风险
- 政策风险
- 市场整体波动风险

### 非系统性风险
- 个股风险
- 行业风险
- 财务风险

## 风险控制原则

### 仓位管理
- 单只股票仓位不超过总资金的X%
- 行业集中度控制
- 分散投资原则

### 止损策略
- 技术止损：跌破关键位置时止损
- 时间止损：超过预期时间未达到目标时止损
- 资金止损：单笔交易亏损达到X%时止损

### 止盈策略
- 目标收益达成时止盈
- 技术位止盈
- 基本面恶化时止盈

## 风险评估方法

### 个股风险评估
- 基本面健康度
- 技术面风险
- 市场情绪影响
- 资金流向情况

### 组合风险评估
- 行业集中度
- 个股关联性
- 整体仓位水平

## 风险控制检查清单

### 买入前检查
- [ ] 基本面是否健康
- [ ] 技术面是否支持
- [ ] 估值是否合理
- [ ] 仓位是否合适
- [ ] 止损位是否明确

### 持有中检查
- [ ] 基本面是否恶化
- [ ] 技术面是否破位
- [ ] 市场情绪是否改变
- [ ] 是否达到止盈/止损条件

### 卖出后检查
- [ ] 卖出理由是否成立
- [ ] 是否存在误判
- [ ] 可以改进的地方
